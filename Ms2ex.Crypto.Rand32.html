    Ms2ex.Crypto.Rand32 (MS2EX v0.1.0)

    View Source

    Implements a 32-bit random number generator used in the MapleStory 2 encryption protocol.

    This module provides functionality to create and manipulate random number states based on seeds, as well as generate random integers and floats from those states.
